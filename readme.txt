How to run the tests

1. Put all your .cpp / .h files in this folder.
2. Make sure the tests/ folder is present.
3. Run:

Linux / macOS:
python3 run_tests.py

Windows:
python run_tests.py

Note: Implement Racenion26b2.h / Racenion26b2.cpp.
The provided main26b2.cpp and wet2util.h must not be submitted.
